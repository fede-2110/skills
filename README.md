# Agent Skills

Backup of Claude Code skills. Lives directly at `~/.claude/skills/`.

## Restore

```bash
git clone https://github.com/fede-2110/skills.git ~/.claude/skills
```

## Layout

- **[engineering/](./engineering/)** — code work skills (review, design, refactor, TDD, triage, PRDs, plans).
- **[productivity/](./productivity/)** — general workflow tools, not code-specific.
- **[misc/](./misc/)** — kept around but rarely used.

Most skills are from [mattpocock/skills](https://github.com/mattpocock/skills); the rest are custom.
